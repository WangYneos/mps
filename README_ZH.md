## 📖 介绍
MPS 是一个中间代理服务扩展库。
它支持 HTTP、HTTPS、Websocket、正向代理、反向代理、中间人等代理方式

## 🚀 特性

- [X] Http代理
- [X] Https代理
- [X] 正向代理
- [X] 反向代理
- [X] 隧道代理
- [X] 中间人代理 (MITM)
- [ ] WekSocket代理
- [ ] Socks5代理

## 🧰 安装

```
go get -u github.com/telanflow/mps
```
